# 🔐 מערכת ניהול שחרור חשבונות

אפליקציית ווב לניהול לקוחות של שחרור חשבונות (וואטסאפ / Google / פייסבוק וכו').
עוקבת אחרי תאריכי שחרור, תשלומים, ומזכירה לך מה דחוף.

---

## מה יש באפליקציה (פאזה 1 — מוכן ועובד)

- **מסך "היום"** — מחלק את הלקוחות לקטגוריות: 🔴 באיחור · 🟠 משתחרר היום · 🔵 בשבוע הקרוב · ⚪ בהמשך
- **סטטיסטיקות למעלה** — כמה חשבונות פעילים, כמה כסף ממתין לתשלום, כמה באיחור
- **ניהול לקוח** — שם, טלפון, סוג שירות, תאריך שחרור, סכום, סטטוס תשלום (ישולם בשחרור / שולם / חלקי), הערות
- **כפתורים מהירים בכל כרטיס:**
  - 💬 **וואטסאפ** — פותח צ'אט עם הלקוח עם הודעה מוכנה
  - 📅 **ליומן** — יוצר אירוע ביומן Google עם תזכורת אוטומטית
  - ✓ **שולם** — מסמן שהתשלום התקבל
  - 🔓 **שוחרר** — מסמן שהעבודה הושלמה
- **חיפוש וסינון** — לפי שם/טלפון, לפי באיחור / לא שולם / הושלמו
- **נעילת קוד** — 4 ספרות (יש שם פרטי לקוחות)
- **גיבוי** — ייצוא/שחזור לקובץ

---

## איך להעלות את זה לאוויר (5 דקות, חינם)

הכי פשוט — **Netlify Drop**:
1. גש ל‑ https://app.netlify.com/drop
2. גרור לשם את **כל התיקייה** `release-manager`
3. תקבל כתובת קבועה (למשל `https://xxx.netlify.app`)
4. פתח את הכתובת ב‑**Safari** באייפון

*(אפשרויות חלופיות: Vercel, Cloudflare Pages, GitHub Pages — כולן חינם.)*

---

## התקנה למסך הבית באייפון

1. פתח את הכתובת ב‑**Safari** (חשוב — לא כרום)
2. לחץ על כפתור השיתוף ⬆️ (למטה באמצע)
3. בחר **"הוסף למסך הבית"**
4. עכשיו יש לך אייקון כמו אפליקציה רגילה, נפתח במסך מלא

## פאזה 2 — כשתרצה להתקדם

הדברים האלה דורשים חשבונות שלך, נסדר אותם ביחד:

### ☁️ סנכרון ענן (נגיש מכל מכשיר + גיבוי אוטומטי)
נחבר **Supabase** (חינם). היום הנתונים על המכשיר בלבד; אחרי החיבור —
תוכל לפתוח מהמחשב, מהנייד, ואם עובד/בן משפחה עוזר לך גם הוא רואה.

### 📲 הודעת וואטסאפ אוטומטית אליך כל בוקר
"בוקר טוב, היום צריך לשחרר 2 חשבונות ולגבות ₪900".
דורש חיבור ל‑**WhatsApp Cloud API** + פונקציה מתוזמנת ב‑Supabase.
זה החלק היחיד שעולה קצת מאמץ/כסף — לכן השארנו לסוף.

### 🔔 התראת push באייפון
אחרי התקנה למסך הבית (iOS 16.4+) אפשר להוסיף התראות דחיפה.
